# Primeiros Passos

```
git clone https://github.com/Luis-F-Oliveira/PClient.git
cd ./PClient
npm install
```

# Arquivo .env

Copie o arquivo `.env.example`

```
cp .env.example .env
```

Depois adicione a rota do seu backend, ex: `http://localhost:0000`

```env
NEXT_PUBLIC_BACKEND_URL=
```

# Construir aplicativo

```
npm run build
npm run start
```
